---
title: TypeScript SDK
description: The official TypeScript/JavaScript client SDK for SparrowDB.
---

The TypeScript SDK (`sparrow-sdk`) provides a query-builder DSL and typed client for calling compiled HQL queries from Node.js and browser environments.

## Install

```bash
npm install sparrow-sdk
# or
pnpm add sparrow-sdk
```

## Usage

```typescript
import { SparrowClient } from 'sparrow-sdk';

const client = new SparrowClient({
  baseUrl: 'http://localhost:6969',
  apiKey: 'sparrow_your_token_here',
});

// Call a compiled HQL query
const result = await client.query('GetUserById', {
  id: '018f2e3a-1234-7abc-8def-000000000001',
});

console.log(result);
```

## Source

The SDK lives in `sdks/ts/` in the SparrowDB monorepo and is published to npm as `sparrow-sdk`. It has zero dependency on internal SparrowDB crates or the Rust toolchain — it only communicates via the HTTP API. Proprietary — see LICENSE for terms.
