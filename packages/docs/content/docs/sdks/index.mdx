title: SDKs
description: Official client SDKs for SparrowDB — Rust and TypeScript.
---

SparrowDB ships two official client SDKs. Both are Apache-2.0 licensed and published as standalone packages with zero dependency on internal SparrowDB crates.

| SDK | Package | License |
|-----|---------|---------|
| [Rust SDK](/docs/sdks/rust) | `sparrow-sdk` on crates.io | Apache-2.0 |
| [TypeScript SDK](/docs/sdks/typescript) | `sparrow-sdk` on npm | Apache-2.0 |

Both SDKs communicate with SparrowDB over its HTTP API and do not require any native dependencies.
