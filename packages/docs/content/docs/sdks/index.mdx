SparrowDB ships two official client SDKs. Both communicate with SparrowDB over its HTTP API and do not require any native dependencies.

| SDK | Package |
|-----|---------|
| [Rust SDK](/docs/sdks/rust) | `sparrow-sdk` on crates.io |
| [TypeScript SDK](/docs/sdks/typescript) | `sparrow-sdk` on npm |
